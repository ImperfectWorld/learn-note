## 无重叠区间
### 描述给定一个区间的集合 intervals ，其中 intervals[i] = [starti, endi] 。返回 需要移除区间的最小数量，使剩余区间互不重叠 ：。

### 示例:
```
输入: intervals = [ [1,2], [1,2], [1,2] ]
输出: 2
解释: 你需要移除两个 [1,2] 来使剩下的区间没有重叠。
```

### 分析
- 找到结束最早的
- 把所有有交集的删除掉
- 重复以上两步
